Setup

Add the CSS inside the <head>-tag of your document

<link rel="stylesheet" href="path/to/css/directory/sliderplugin.css">

Add the required mark up to the <body> of your document

<!-- Required mark up -->
<div class="slider">
    <ul class="slide-wrapper">
        <li><div class="slide"><img src="slides/1.jpg"></div></li>
        <li><div class="slide"><img src="slides/2.jpg"></div></li>
        <li><div class="slide"><img src="slides/3.jpg"></div></li>
        <li><div class="slide"><img src="slides/4.jpg"></div></li>
    </ul>
    <!-- The navigation can be placed anywhere in the document -->
    <div class="slide-nav">
        <a href="#" class="prev" data-dir="prev">Previous</a>
        <a href="#" class="next" data-dir="next">Next</a>
    </div>
    <!-- The dots / numbers are entirely optional -->
    <div class="slide-dots"></div>
</div>

Note: In this example, the navigation and slide-dots have been placed in the <slider>-container. They can however be placed anywhere in the document.

Add the script to your page, either in the <head> or as the last item in the <body>. You will also need to add the jQuery library to your document.

<script src="path/to/js/directory/jquery.js">
<script src="path/to/js/directory/sliderplugin.js">

Using the plugin

In your main javascript file you will need to create a new instance of the Slider-object. The Slider constructor function takes three arguments:

  • container - the element that wraps the list of images.
  • nav - the element that wraps the "prev" and "next" links
  • dots - the div that will be populated with the "dot-style" links. Optional

//Create a new instance of the Slider
var slider = new Slider( $(".slide-wrapper"), $(".slide-nav"), $(".slide-dots") );

//Initialize the new instance
slider.init();

Next you will need to set up the event listeners

//Previous and Next Links
$(".slide-nav").on("click", "a", function(e) {
    e.preventDefault();

    //The currentSlide function takes the data-dir attribute on the links as its argument
    slider.currentSlide( $(this).data("dir") );
    
    //Set the active class on the dots
    slider.setActiveDot();

    //The move function takes a prefix as its argument, use px, %, em or whatever you want
    slider.move("px");
});

//Navigation dots / numbers
$(".slide-dots").on("click", "a", function(e) {
    e.preventDefault();

    //Use the data-slide attribute as argument
    slider.setSlide( $(this).data("slide") );


    slider.setActiveDot();
    slider.move("px");
});
